Doble Programa y Articulación con el Posgrado

Rutas de formación para optimizar el tiempo de estudio y cumplir requisitos de doble programa

Después de un análisis con cada proyecto curricular, se ha diseñado una ruta de formación para el doble programa, donde el estudiante puede completar los requisitos del proyecto de Ingeniería de Sistemas a través de espacios de formación obligatorios y electivos.

Esta ruta permitirá homologar el ciclo básico y la formación en Sistemas al segundo proyecto, reduciendo el tiempo para completar ambos programas. Por ejemplo, un estudiante de Sistemas podría realizar un doble programa en Electrónica, completando una ruta en Control para cumplir los requisitos de ambos proyectos de manera optimizada.


Articulación con el Posgrado

Para facilitar el avance hacia el posgrado, el estudiante puede tomar cursos de posgrado que cumplan los requisitos de Electivos RF y Flexibles RF, dentro del cuerpo de conocimiento de su Ruta de formación seleccionada. Estos cursos podrán ser homologados al ingresar al posgrado, reduciendo así el tiempo de permanencia en el mismo, conforme a las reglamentaciones institucionales.
